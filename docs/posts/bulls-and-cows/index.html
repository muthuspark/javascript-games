Bulls and Cows - Number Guessing Game

Guess the secret number using logic and deduction. Bulls mean correct digit in correct position, cows mean correct digit in wrong position.

FAQ

Try to guess the secret number. After each guess, you get feedback: Bulls (üéØ) mean you have a correct digit in the correct position. Cows (üêÑ) mean you have a correct digit but in the wrong position. Use this feedback to narrow down the secret number.

The secret number has unique digits (no repeats) and does not start with 0. You must guess a number with the same properties. You have 10 attempts to guess correctly. Each guess shows how many bulls and cows you scored.

Start with digits spread across the range (like 1234) to identify which digits are in the secret. Track which digits gave bulls vs cows. Once you know some correct digits, focus on finding their positions by moving them around in subsequent guesses.
